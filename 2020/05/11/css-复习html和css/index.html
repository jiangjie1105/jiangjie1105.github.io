2.css权重

  • css权重主要是由数值决定的，具体对应数值如下：
    ！important Infinity

    1. 行间样式1000
    1. id ：100
    1. class|属性|伪类 ：50
    1. 便签|伪元素： 1
    1. 通配符： 0

  • 举例如下：

    <!DOCTYPE html>
<html>
<head>
  <title>css权重</title>
  <meta charset="utf-8">
</head>
<body>
  <div>
      <p>css权重</p>这个权重值为：1+1=2 <!-- 这个权重值为：1+1=2 -->
  </div>
  <div>
      <p id="lei">css权重</p>        <!-- 这个权重值为：1+100=101 -->
  </div>
</body>
</html>

    再举个例子：

    <!DOCTYPE html>
<html>
<head>
  <title>css权重</title>
  <meta charset="utf-8">
  <style type="text/css">
      p{
          font-size: 20px;
          color: red;
      }
      div p{
          font-size: 15px;
          color: green;
      }
      #lei{
          font-size: 25px;
          color: yellow;
      }
  </style>
</head>
<body>
  <div>
      <p id="lei">css权重</p>
  </div>
</body>
</html>

    上面的例子是：三条样式修饰同一元素p,那么最终的样式如何呢？很显然，最后的样式是第三个样式也就是字体大小为25px,颜色为yellow。

2.标签的作用：

作用一：外部页面链接：

<a href="http://www.baidu.com" target="_blank>百度</a>
  • 1）：这里的href规定的是指向页面的url地址
  • 2）：target指的是跳转网页的方式。target有四个属性。分别为 _blank,_parent,_self,_top.

其中：1. _blank代表在新窗口打开页面的链接地址，如果设置则另起网页跳转;

  1. _self代表在自身窗口打开页面链接，默认为self;

  2. _parent代表在父窗口打开此网页；

  3. _top代表的是整个窗口中打开此网页，测试结果与self相同；

作用2：本地页面链接：

在同一个项目中，href的指向为相对路径，此处的node文件和本文件处在同一个项目中，所以直接写名字即可。

<a href="node.html">跳转node.html文件</a>

超链接跳转有三个路径

  1. 绝对路径：从文件所在的最高级目录下开始的完整路径。
  • 1.1：访问互联网上的资源。
  • 1.2：访问本机的资源。
  1. 相对路径： 从文件所在的位置开始查找，资源文件所在的位置经过的路径。
  • 2.1: 当文件和资源文件在同一目录下直接可以用名称引用。
  • 2.2：资源文件在当前目录的子文件中，先进入，再引用。
  • 2.3：资源文件再当前文件的父目录中，先返回，再引用。
  1. 根相对路径：在服务器上使用的，只有服务器端才可以使用。
