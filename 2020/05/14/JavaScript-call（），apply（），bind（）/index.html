以上除了bind方法需要加（）外，结果返回都是一致的，由此可以得出结论，bind返回一个新的函数，必须调用它才能执行。

2. 对比call(),apply(),bind()传参情况下;

  • 不难看出，三者有着微妙的差距，三者括号中第一个参数都是this的指向对象，第二个参数的差别就来了，
  • call()的参数是直接放进去的，第二个第三个到第n个参数全部用逗号隔开，直接放到后面;obj.myFun.call(db,’西安’, … ,’string’ )；
  • apply的所有参数都必须放在一个数组里面传进去：
    obj.myFun.apply(db,[‘西安’, …, ‘string’ ]);
  • bind除了后面要加一个括号调用外，参数和call是一样的。
  • 三者的参数并不都限制于string类型，也允许是各种类型，如函数，obj等
