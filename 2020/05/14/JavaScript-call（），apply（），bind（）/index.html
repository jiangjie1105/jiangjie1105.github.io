JavaScript-call（），apply（），bind（）

2020-05-14
javaScript
Word count: 533   |   Reading time≈ 2 min

先看以下例子
例一：

var name = "小王",age = 17;
var obj = {
    name:"小张",
    objAge:this.age,
    myFun:function(){
        console.log(this.name + "的年龄" + this.age);
    }
}
obj.objAge;
obj.myFun();

例二：

var name = "小姜";
function foo(){
    console.log(this.name);
}
foo();//小姜

比较一下二者的区别可以发现，例一中的this指向的是obj，例二的this指向的是全局window；

1. call(),apply(),bind()都是用来重新定义this这个对象的,如：

var name = "小王",age = 17;
var obj={
    name:"小张",
    ObjAge:this.age,
    myFun:function(){
        console.log(this.name + "的年龄" + this.age);
    }
}
var db = {
    name:'小姜',
    age:6
}
obj.myFun.call(db);
obj.myFun.apply(db);
obj.myFun.bind(db)();

//小姜的年龄6
//小姜的年龄6
//小姜的年龄6

以上除了bind方法需要加（）外，结果返回都是一致的，由此可以得出结论，bind返回一个新的函数，必须调用它才能执行。

2. 对比call(),apply(),bind()传参情况下;

var name = "小王",age = 17;
var obj={
    name:"小张",
    ObjAge:this.age,
    myFun:function(fr,t){
        console.log(this.name + "的年龄" + this.age + "岁来自" + fr + "去往" + t);
    }
}
var db = {
    name:'小姜',
    age:6
}
obj.myFun.call(db,'西安','上海');
obj.myFun.apply(db,['西安','上海']);
obj.myFun.bind(db,'西安','上海')();
//小姜的年龄6岁来自西安去往上海
//小姜的年龄6岁来自西安去往上海
//小姜的年龄6岁来自西安去往上海
  • 不难看出，三者有着微妙的差距，三者括号中第一个参数都是this的指向对象，第二个参数的差别就来了，
  • call()的参数是直接放进去的，第二个第三个到第n个参数全部用逗号隔开，直接放到后面;obj.myFun.call(db,’西安’, … ,’string’ )；
  • apply的所有参数都必须放在一个数组里面传进去：
    obj.myFun.apply(db,[‘西安’, …, ‘string’ ]);
  • bind除了后面要加一个括号调用外，参数和call是一样的。
  • 三者的参数并不都限制于string类型，也允许是各种类型，如函数，obj等
