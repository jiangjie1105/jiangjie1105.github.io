1.js函数

1. 函数定义

函数使用function定义，后跟一组参数及函数体 ，语法如下

function functionName([arg]){

}//arg为函数的参数，不是语法

如

<script>
    function myFun(){
        alert("我是一个函数");
    }
    myFun();
    </script>

函数必须经过调用才能使用

2.函数的返回值

function add(num1,num2){
    var sum = num1 +num2;
    return sum;
}
console.log(add(2,3));
alert(add(23,33));

  1. 函数会在执行完return语句之后立即停止或退出

  2. return并不一定要有返回值

    3.arguments

  3. ECMAScript中的参数在内部用一个数组表示，在函数体内部通过arguements对象来访问这个数组参数。

说明1.arguements对象只是与数组类似，并不是array的实例

9.slice（）

语法：arrayObject.slice（start,end）

  • 掌握使用splice（）方法删除数组项

  1. arrObject.splice(index,count)
    功能：删除从index处开始的零个或多个元素

  2. 返回值：含有被删除的元素的数组。

  3. 说明：count是要删除的项目输血量 如果设置为零，则不会删除项目，如果不设置，则删除从index开始的所有值

//删除
        var delArr = arr.splice(2, 0);
        console.log(arr);//输出六个值
        console.log(delArr);//输出零个值
  • 掌握使用splice（）方法插入数组项
 //插入
        var insertArr = arr.splice(2, 3, "m", "n")
        console.log(arr);//输出六个值
        console.log(insertArr);//输出零个值
  • 掌握使用splice（）方法替换数组项
var replaceArr = arr.splice(1,2,"x","y","z")
        console.log(replaceArr);

    返回值：查找的项在数组中的位置

没有找到的情况下返回-1。

var nums = [1,2,3,4,23,2,35,23]；
var pos = nums.indexOf(3,x)；
//这个x指从数组中索引为几开始算，若不填则为0

11.lastIndexOf()

  • 功能：从数组的末尾开始向前查找
    返回值：查找的项在数组中的位置

var nums = [1,2,3,4,23,2,35,23]；
console.log(pos);
//封装一个方法实现indexOf的功能

<script>
    var nums = [1, 3, 4, 5, 7, 8, 45, 43, 22, 11, 35];
    function ArrayIndexOf(arr, value) {
        //检测value在arr中出现的位置
        for (var i = 0; i < arr.length; i++) {
            if (arr[i] === value) {
                return i;
            }
        }
        return -1;
    }
    var pos=ArrayIndexOf(nums,43);
    console.log(pos);//输出为7
</script>

2.内置对象–String

  1. charcodeat 或者字符串中的字母

2. indexOf（）

语法：stringObject.indexOf(“o”)
功能：从一个字符串中搜索给定的字符串位置，返回字符串的位置。
返回值：数值
说明：如果没有找到该字符串，则返回-1

<script>
    var email = "102130103@qq.com";
    console.log(email.indexOf("q"));
    </script>//输出10。（检测第一次出现的位置）

2. lastindexOf（）
lastIndexOf方法返回给定元素在数组中最后一次出现的位置，如果没有出现则返回-1.

注意，以上两种方法不能来搜索nan的位置

3. slice()

语法：
stringObject.slice(start,end)
功能：截取子字符串
参数说明：1.start：必须，指定字符串开始位置

2.end：可选，表示字符串到哪里结束，end本身不再截取范围之内，省略时截取至字符串的末尾，
最后一个参数时 截取到末尾。

3.当参数为负数时，会将传入的负值与字符串的长度相加

<script>
        var url = "http://www.hao123.com/index.txt";
        function getFileFormat(url) {
            //获取. 在url中的位置
            var pos = url.lastIndexOf(".");

        }
        var formatName = getFileFormat(url);
        var picFormat = getFileFormat("35672153.png");

    </script>
